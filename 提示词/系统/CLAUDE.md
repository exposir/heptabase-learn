# 提示词/系统/

> L2 | 父级: 提示词/CLAUDE.md

## 模块定位

本模块包含 AI 系统级提示词（System Prompts）的归档与深度分析，包括完整的系统提示及相关工具（Antigravity, ClaudeCode 等）的运行机制与指令剖析。

## 成员清单

- **Antigravity工具深入分析.md**: 分析 Antigravity 代理系统的内置工具与指令架构。
- **ClaudeCode工具深入分析.md**: 分析 Claude Code 工具体系的工作原理与代理协作机制。
- **System Prompt 深度分析报告.md**: 对大模型系统提示词进行结构解析和核心认知原理提炼。
- **完整系统提示词\_中文.md**: 系统预设提示词的中文翻译与参考。
- **完整系统提示词\_英文.md**: 系统预设提示词的英文原版参考与存档。

## 法则

**解构·洞察·复用**

- **解构**：拆解优秀的系统提示指令成分。
- **洞察**：洞察模型执行背后的认知架构设计（如思维路径和工具定义）。
- **复用**：沉淀高效的指令块以供其他 System Prompt 借鉴。
