# 编程题/

> L2 | 父级: /CLAUDE.md

## 模块定位

本模块归档 **前端及全栈开发的实战编程题与系统设计方案**。不仅关注代码实现，更关注方案对比、边界处理及底层原理。

## 成员清单

- **1.1 JavaScript 核心手写实战.md**: JavaScript 原型、继承、this 绑定、柯里化及深拷贝等语言底层机制的手写实现。
- **1.2 异步编程专题.md**: Promise A+ 规范实现、并发控制、任务调度及异步控制流。
- **1.3 数据处理与工具函数.md**: 数组扁平化、对象深层路径访问、大数运算及常用工具函数。
- **1.4 设计模式实现.md**: EventEmitter、观察者、单例、策略等前端常用设计模式的手写实战。
- **1.5 DOM 与事件.md**: 事件委托、Ajax 封装、视口判定等 DOM 操作与浏览器事件处理。
- **2.2 状态与数据同步的核心挑战.md**: 深度剖析多标签页同步、乐观更新、实时协同及表单持久化四大场景。
- **2.3 大数据与性能场景.md**: 针对 10 万级数据的虚拟滚动、时间分片、Worker 线程及双向加载优化方案。
- **2.4 文件处理场景.md**: 针对 GB 级文件上传与下载的工业级解决方案，涵盖分片、Hash、秒传与流式处理。
- **2.5 实时通信场景.md**: 高可用实时系统的前端实现规范，包含心跳、退避重连与消息可靠性。
- **2.6 安全与稳定性场景.md**: 前端纵深防御体系，涵盖幂等设计、混合加密与防篡改水印方案。
- **3.1 复杂交互与动画.md**: 手势系统、动画编排、拖拽排序与移动端滚动问题的深度方案。
- **3.2 音视频与媒体.md**: 视频播放器封装、音视频录制、WebRTC 实时通信与图像处理。
- **3.3 编辑器场景.md**: 富文本编辑器、代码编辑器集成与 Markdown 编辑器的核心设计。
- **3.4 可视化与图形.md**: Canvas 性能优化、图形编辑器与大数据可视化技术方案。
- **3.5 工程架构.md**: Monorepo 管理、组件库设计、CLI 脚手架与 AST 应用实践。
- **3.6 组件与 SDK 设计.md**: 表单引擎、权限控制、微前端容器与低代码引擎的架构设计。
- **3.7 监控与可观测性.md**: 错误监控 SDK、性能监控指标与埋点 SDK 的设计实现。
- **3.8 跨端场景.md**: JSBridge 设计、离线包方案与多端适配策略。
- **3.9 Node.js BFF.md**: BFF 层设计、SSR 同构渲染与 Serverless 应用优化。
- **3.10 新技术方向.md**: WebAssembly、PWA 与 Web Components 前沿技术探索。
- **前端工程师场景题与编程题完整图谱.md**: P6/P7 前端工程师场景题与编程题的完整知识图谱，四层能力模型总纲。
- **4.1 TypeScript 类型编程.md**: TypeScript 类型编程，涵盖工具类型实现与类型体操实战。
- **4.2 框架原理深度.md**: 框架原理深度，对比 React Fiber 架构、Vue 响应式机制及通用框架原理。
- **4.3 浏览器原理应用.md**: 浏览器原理应用，深入事件循环实战与渲染原理优化。
- **4.4 调试与问题排查.md**: 调试与问题排查，涵盖性能分析、内存泄漏定位及线上问题追踪。
- **4.5 稳定性与质量.md**: 稳定性与质量，探讨前端容灾、测试策略及灰度发布方案。
- **4.6 移动端专项.md**: 移动端专项，解决经典兼容性问题与 Hybrid 开发调试技巧。
- **4.7 安全专项.md**: 安全专项，剖析 XSS/CSRF 等常见攻击与防御体系。
- **4.8 算法在前端的实际应用.md**: 算法在前端的实际应用，汇总虚拟列表、Diff 算法等核心场景。
- **4.9 国际化与可访问性.md**: 国际化与可访问性，i18n 方案与 A11y 最佳实践。
- **图谱补充：面试必考遗漏题.md**: 完整图谱遗漏的 10 道面试必考编程题，按重要性排序，含完整实现。
