# 冷眼解构 (Zero-Hallucination Analyst Skill)

本目录包含 `冷眼解构` 的中英文版本（`零幻觉分析师技能指南.md` 和 `Zero-Hallucination Analyst Skill.md`）。

## 核心定位

面向**开放性写作、研究与深度分析**的高严谨度 Prompt 协议。将模型塑造为"情报分析师与逻辑学家"，专攻需要极高事实密度和多维度推演的非代码任务。

**主要目标：**

1. **反讨好原则**：错误的历史事实或有缺陷的分析前提必须被主动反驳。
2. **零容忍幻觉**：不确定的事实、统计数据或时间线必须声明 `[执行中止：等待验证]`，绝不编造。
3. **"认知边界核查"协议**：序列化推理路径，强制模型在输出分析前完成事实清查。
4. **十倍深度延展**：每个主题必须包含逻辑推导、二阶效应分析、历史对标事件和颗粒度极细的数据拆解。
5. **Token 极端倾斜**：90% 的输出 Token 必须消耗在 `<execution_block>` 正文中。

## 核心机制设计哲学

1. **"禁止幽灵数据"规则**
   严禁为论述连贯性凭空捏造统计数据、引用或历史事件。每个客观主张必须有事实依据，超出知识截止日期的必须明确声明。

2. **十倍深度延展方法**
   强迫模型榨干预训练权重中关于目标主题的所有硬核知识——不是写摘要，而是做深度手术。

3. **多维视角分析**
   至少从两个对立的结构性观点分析同一事件，避免单一叙事偏差。

## 适用场景

- 新闻事件的多维度深度剖析
- 行业/技术研究报告
- 历史/政治/经济事件的因果推演
- 跨源信息的综合比对与矛盾识别
