# 界面美学 (High-Agency Frontend Skill)

本目录包含 `界面美学` 的中英文版本（`高能动前端技能指南.md` 和 `High-Agency Frontend Skill.md`）。

## 核心定位

所有 Skill 中**体量最大、最专业化**的一个，专为**前端 UI/UX 开发**设计。一套完整的**设计工程操作系统**——通过量化参数驱动、AI 偏见纠正矩阵和 100 条禁止模式清单，强制模型输出达到高端设计工作室水准的前端代码。

**主要目标：**

1. **量化设计参数**：通过三个核心刻度盘（`DESIGN_VARIANCE=8`、`MOTION_INTENSITY=6`、`VISUAL_DENSITY=4`）精确控制输出风格。
2. **AI 偏见纠正**：系统性消灭 LLM 的 UI 陈词滥调（居中布局偏见、Inter 字体依赖、三列等宽卡片、紫色 AI 美学等）。
3. **100 条禁止模式**：明确封堵 LLM 常见的"AI 设计特征"（霓虹光晕、纯黑色、过饱和色、通用名称、假数据等）。
4. **创意武器库**：40+ 高端交互范式（磁性按钮、液态玻璃、视差倾斜卡片、水平滚动劫持等）。
5. **性能护栏**：强制硬件加速动画、DOM 开销控制和 Z-Index 克制。

## 核心架构

```
基线配置 (§1)    → 量化三刻度盘
架构规范 (§2)    → React/Next.js/Tailwind 技术约束
设计工程 (§3)    → 排版/色彩/布局偏见纠正
创意主动性 (§4)  → 液态玻璃/磁性物理/持续微交互
性能护栏 (§5)    → GPU 加速/DOM 成本/Z-Index
技术参考 (§6)    → 三刻度盘完整定义
禁止模式 (§7)    → 100 个 AI 特征黑名单
创意武器库 (§8)  → 40+ 高端交互范式
Bento 范式 (§9)  → SaaS Dashboard 动效引擎
起飞检查 (§10)   → 最终代码质量矩阵
```

## 适用场景

- 高端 Landing Page / Marketing Site
- SaaS Dashboard 与数据可视化界面
- 产品 Feature Section 与 Bento Grid
- 需要动效编排的交互式 UI 组件
