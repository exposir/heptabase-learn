# Gemini CLI 更新日志 (Changelog)

> 本文件整理并维护自 [Gemini CLI changelogs](https://geminicli.com/docs/changelogs)。

## 版本发布渠道 (Release Channels)

Gemini CLI 主要包含以下发布渠道：

- **Nightly**: 包含最新变更的每夜构建版。
- **Preview**: 用于接收早期反馈的实验功能版。
- **Stable**: 稳定的推荐版本，适合日常使用。

---

## 版本记录

### v0.30.0 - 2026-02-25

- **常规维护与升级**: 常规架构优化与核心依赖升级至 0.30.0，包含常规的 bug 修复与性能提升。

### v0.29.0 - 2026-02-17

- **Plan Mode**: 添加全新综合规划能力，包含 `/plan` 辅助、`enter_plan_mode` 工具及配套专属文档。
- **Gemini 3 默认化**: 移除 Preview 标志，默认为所有用户开启启用 Gemini 3 模型架构。
- **扩展探索与管理**: 新一代 UI 视图并允许直观管理扩展设置。
- **管理员控制域 (Admin Control)**: 管理员现可通过专属配置将指定 MCP 伺服纳入预设白名单。

### v0.28.0 - 2026-02-10

- **Slash Command**: 添加了新的 `/prompt-suggest` 支持动态生成系统提示词。
- **IDE 支持**: 正式支持并集成 Positron IDE。
- **Customization (环境定制)**: 支持根据终端底层颜色自动切换系统显示主题，并在扩展中启用了自定义界面颜色的设计。
- **Authentication 配置增强**: 支持 OAuth 协议下的交互式 / 非交互授权操作，在 bug 呈报中自动附加所采用认证模式。

### v0.27.0 - 2026-02-03

- **架构更替 (Event-Driven Architecture)**: 引入新事件驱动核心引擎与调度器来管理工具任务。
- **交互模式大革新**: 加入可展开阅览的大量文本框粘贴能力以及排队触发工具确认视图，大幅精简工作流中断频率。
- **新功能命令**: `/rewind` 快速调阅或重置之前会话历史。
- **平台性加强**: 完全支持搭载 Linux Wayland 和 X11 下图像剪贴板拷贝粘贴支持。

### v0.26.0 - 2026-01-27

- **Agents + Skills (引擎体系)**: default 状态下预设开启 agent skills 并提供新引入工具 `skill-creator`。添加全能代理 `generalist agent` 并极大改良系统调度分派任务之智能路由。
- **开发者辅助增强**: 推出了全新 `/introspect` 内联系统查询用作核心级别运行状态调校控制面板。
- **稳定性与防错处理**: 重构处理多字节字符崩溃处理以及修正代理任务无法分配权限等长期历史性 Bug。

### v0.25.0 - 2026-01-20

- **代理能力升级**: 支持执行创建 PR 动作指令插件 (pr-creator)，丰富系统级内建辅助。
- **前端 UI**: `Tab` 键实现智能聚焦 Shell 以及 Input 层区域；更透明展现出 agent 调用的控制面板情况。
- **效能增强**: 实现了对于 `.geminiignore` 内容规则系统读取级缓存提升并发表现，增强错误重试处理方案 (Retry mechanism)。

### v0.24.0 - 2026-01-14

- **安全态势改进 (Security)**: `Policy Engine` 大升级，提供极细力度管理方案拦截 shell；默认调整本地执行权限由 Trusted 信任化降为不可信环境以保护系统。
- **底层改造的 Third Phase**: 强绑定 MessageBus 控制执行总线与所有代理沟通信号传递体系。
- **Remote Subagents 配置体系**: 开拓式容许分配远端资源环境完成部分计算处理以组成分散且强大的本地自动化任务网。

### v0.23.0 - 2026-01-07

- 🎉 **初体验 Agent Skills**: 首发试作预览了技能代理体系架构功能。
- **统计工具指令**: 新加入了类似年终统计数据的 `npx gemini-wrapped` 表现汇总展示。
- **控制增强选项操作**: 添加了对 Window 底下利用 `Alt+V` 即可处理图像流直贴输入环境支持。

### v0.22.0 - 2025-12-22

- 🎉 **开放权限解脱限制**: `Free Tier` (免费层级) 已能经由选单开启使用到最强大模型之 Gemini 3 Pro 级或 Flash 基础功能权限。
- **生态环境插件版图扩大并包含更精细代码和系统诊断与支持服务应用扩展环境等包括了 Conductor 及 Endor Labs**。

### v0.21.0 - 2025-12-15

- ⚡️ **引入 Gemini 3 Flash 生态体系以取代传统高耗时 2.5 系列核心网络以最大程度带来提速增效降低算力花销使用**。
- **原生集成资源检索**: 引流借由打出了专门用于定位操作解析功能系统 `@` 以精确定位资源与模块并定位。

### v0.20.0 - 2025-12-01

- **新增导入拖曳功能文件支持解析以及安全拦截执行持久性机制记录保存操作功能以提升自动化应用效力**。

### v0.19.0 - 2025-11-24

- **环境控制和交互体系优化调整（如增修点触激活 Click to Focus 等交互改进）并在扩增如与 Zed IDE 强关联及音频制作扩展套件处理管理执行功能支持设定系统操作使用配置选项改善体验支持设计**。

_(在涵盖版本 v0.18 及以下等旧版时期内，主要涉及：系统互动 UI 及底层调度、各类 Google Workspace (文档服务套件等)，各个类型的主流数据库如 BigQuery/PostgreSQL MCP 等整合体系搭建等，具体发展史可参考原始 Changelog 更新节点详细资料。)_
